Welcome to Circles!

Circles is currently in an open alpha. It's a buggy mess, but that's okay! If you find any bugs, please report them here. There's plenty of things that need fixed or implemented, so if you're interested, help out and make a contribution!

If you're viewing this on the production server, I know how slow it is. I'm paying the minimum possible and my code is very unoptimized. If more people are interested in this project, I'll scale everything up a bit.

Also, while the UI is now mostly supported on mobile, touch events are not yet working. If you want this to get implemented faster, drop a comment

A big thanks to all of the contributors who've already helped out, and thank you for giving Circles a try!
