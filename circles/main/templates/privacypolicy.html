Privacy Policy

Most documents like this is a mess of legal terms that the normal person doesn't have the time or patience to read. Here, I'm clearly describing what data is collected, how it's stored and what we do with it. In plain English. Nothing else.

What data is collected?

Below I've created categories for each kind of item in Circles. From a User account to a Message, this is what data is assigned to each item.

User

  • Username
  • Password (Hashed)
  • Email
  • Display Name
  • Current x and y position
  • Online status
  • Server and Circle location
  • Selected Conversation and Conversation type
  • Followers
  • Account Created date

Circle

  • Name of Circle
  • Parent Circle or Server
  • x and y position
  • The User that created the Circle

Conversation

  • The name of the Conversation
  • The date and time when the Conversation was created
  • The Users in the Conversation

Message

  • The contents of the Message
  • The date and time when the Message was created
  • The User that created the Message
  • The Conversation or Circle that the Message is in
  • The current type of the Message

What data is shared with others?

Right now, there aren't any other services that use any of your data. In future, when the ability to connect your Circles account with other services and servers becomes available, this section will be updated.

What data is sold to others?

Nothing. Ever. If you're worried you could self host once the necessary infrastructure is finished.
